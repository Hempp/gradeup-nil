# NEXUS-PRIME Project Configuration

This project uses NEXUS-PRIME v4.6.0 orchestration system.

## Installed Modules
- **Core**: Foundation agents (ARCHITECT, SENTINEL, VELOCITY, GENESIS, ORACLE)

## Available Commands
- `/nexus` - Activate NEXUS-PRIME
- `/nexus-status` - Show system status
- `/nexus-help` - Display help

## Quick Deploy
Add more capabilities by installing modules:
```bash
nexus-deploy install stack-prime    # Full-stack infrastructure
nexus-deploy install forge-x        # Software engineering
nexus-deploy install innovation     # 11-Star + Coordinator
```

## Configuration
Edit `.nexus/config.yaml` to customize your setup.
